290  /**

293  * Applies following rules which are equal to the sorting rules of sqlite.
294  *
295  * - Type::Null is alwasy less than any other storage type.

297  * -- if the type is different, but the value equal,
298  * than Type::Int < Type::Real .
299  * if this is unwanted, use the weak_lt function.
300  * -- always less than Type::Text or Type::Blob
301  * - Type::Text is less than Type::Blob
302  *
303  * The comparison of the value itself is implemented via std::less.
304  *
305  * \param a first element to compare
306  * \param b second element to compare
307  *
308  * \returns true if given Value a is less than given Value b
309  */
