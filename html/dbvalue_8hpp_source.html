26  /**
27  * \brief
28  *
29  * This class models the duck typing sqlite uses.
30  * It supports int, real, text, blob and null values.
31  *
32  * The type can be a variant, to support all of the possible types,
33  * or it can be set explicit to a wanted type.
34  *
35  * Using this type for communication with the database can ensure either
36  * type guarantee or give the flexibility of variant types.
37  *
38  * If a type is of type variant, the storage type can be any of the
39  * supported types, otherwise, if the type is set to a specific type, it
40  * is guaranteed that the type will match, or an exception occurs.
41  *
42  */
