◆ dbval_type_lt()

bool sl3::dbval_type_lt ( const DbValue &  a,
const DbValue &  b 
)
noexcept

less than, including type info

Applies following rules which are equal to the sorting rules of sqlite.

  • Type::Null is always less than any other storage type.
  • Type::Integer or Type::Real is always less than Type::Text or Type::Blob

  • Type::Text is less than Type::Blob

    The type used is DbValue.getStorageType.

    The comparison of the value itself is implemented via std::less.

Parameters
afirst value to compare
bsecond value to compare
Returns
true if given DbValue a is less than given DbValue b
